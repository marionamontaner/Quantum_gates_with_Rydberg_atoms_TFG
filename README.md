# Quantum gates with Rydberg atoms

Here you can find the Python code I developed for my Bachelor's thesis (*Treball de Fi de Grau*) at the Universitat de Barcelona.

The project consists of a Jupyter Notebook that combines the code used for the simulations with theoretical explanations.
Inside, you will find the models to simulate single-qubit operations, the implementation of a two-qubit CNOT gate using the Rydberg blockade mechanism,
and a practical simulation of a dense coding quantum communication circuit.
